# Python Utilities Repository

A simple Python repository containing utility functions and calculator operations.

## Files

- `utils.py` - Utility functions including greeting, string manipulation, and number operations
- `calculator.py` - Basic calculator functions for mathematical operations

## Usage

### Utils Module

```python
from utils import greet, add_numbers, reverse_string, is_even

print(greet("Alice"))
print(add_numbers(5, 3))
print(reverse_string("Python"))
print(is_even(4))
```

### Calculator Module

```python
from calculator import add, subtract, multiply, divide, power

print(add(10, 5))
print(subtract(10, 5))
print(multiply(10, 5))
print(divide(10, 5))
print(power(2, 3))
```

## Requirements

- Python 3.6+
